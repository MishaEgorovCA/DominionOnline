Optional card images (private / non-distributed)
================================================

For face-up art, add one file per card id, one of:
  {cardId}.png
  {cardId}.webp
  {cardId}.jpg
  {cardId}.jpeg

Example: village.png, copper.png

The client resolves in that order. Missing files show the card name as a fallback.

Optional face-down pile / deck back: _back.png (or .webp, .jpg, .jpeg)

Do not commit or redistribute official Dominion artwork; use this folder only
for your own private assets.
